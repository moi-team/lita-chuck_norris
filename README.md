# moiscot-problem

MOi Solutions problem bot

## Installation
Add lita-chuck_norris to your Gemfile:

``` ruby
gem 'moiscot-problem'
```

## Usage
    Lita: problem

## Build (development)

    gem build moiscot-problem.gemspec

## Publish (development)

  gem push moiscot-problem-x.x.x.gem
